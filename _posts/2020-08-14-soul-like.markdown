![]({{site.baseurl}}/img/2020-08-14-erro.jpg)

解决方案

修改配置文件`IDA 7.0\cfg\hexrays.cfg`

```assembly
MAX_FUNCSIZE            = 64        // Functions over 64K are not decompiled

// 修改为：
MAX_FUNCSIZE            = 1024        // Functions over 64K are not decompiled
```
