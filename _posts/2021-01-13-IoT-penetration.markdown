### 固件逆向之敏感函数/代码段回溯

合理借助反汇编工具（IDA/Ghidra）脚本

列出引用system的函数地址

```assembly
import ida
import idc
system_list = set()
for loc,name in Names():
    if "system" == name:
        for addr in XrefsTo(loc):
            system_list.add(addr.frm)
print(system_list)
```
