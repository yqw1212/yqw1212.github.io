可以看出，将输入的字符串（flag）的花括号间的内容经过第一次循环中的变化后，与已经给出的v15~v30变量比较必须相同。

而在第一次循环中，根据判断语句的条件，只改变了输入的字符串中的字母，即如果输入的字符非字母则不改变。

由此编写C语言脚本

```assembly
#include <stdio.h>
#include <stdlib.h>

int main()
{
    int a[16] = {81,115,119,51,115,106,95,108,122,52,95,85,106,119,64,108};
    for(int i=0;i<16;i++)
    {
        int flag = 0;
        for(int j=65;j<=122;j++)
        {
            int k=j;
            if(k>64 && k<=90)
            {
                k = (k-51)%26+65;
            }
            if(k>96 && k<=122)
            {
                k = (k-79)%26+97;
            }

            if(k == a[i]){
                printf("%c",j);
                flag = 1;
                break;
            }
        }
        if(flag == 0)
        {
            printf("%c",a[i]);
        }
    }
    return 0;
}
```
